# Goodcase 记录

当前第一版 goodcase 候选墙来自用户偏好的样例和公开图形模式抽象。使用它作为选择图形结构的视觉菜单。

资产：
- `assets/structured-expression-goodcase-wall.svg`

## 已沉淀的用户偏好

- 用户主要需要 PRD / 业务流程图、数据分析图、竞品分析图、战略汇报图、知识地图等结构化表达图。
- 最大痛点：不知道该用什么图，以及画图太慢。
- 偏好风格：咨询公司风格，简约，不要太多颜色，但重点要突出。
- 偏好输出：飞书文档、PPT、draw.io、Mermaid、SVG 预览。
- 用户希望先看到 2-3 个推荐方向，再选择最终图形。
- 可编辑源格式比静态美观更重要。

## 第一版候选墙

已选择纳入的 12 个模板方向：

- A1 用户主路径流程图。
- A2 角色泳道流程图。
- A3 页面 / 状态流转图。
- B1 指标拆解树。
- B2 漏斗诊断图。
- B3 留存 / 分群热力图。
- C1 竞品能力矩阵。
- C2 定位四象限。
- C3 市场 / 生态地图。
- D1 战略路线图。
- D2 能力模型 / 冰山分层。
- D3 全景信息地图。

当用户补充更多样例时，按下面格式更新本文件：

```markdown

- 对应模板：
- 输出格式：
```
